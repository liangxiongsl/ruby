# 4.哈希/散列`Hashes` #

- 实际上就是映射`map`
- 键值对以`=>`分隔，键和值之间以`,`分隔，所有键值对被`{}`包围
- 引用方式：`<变量>[<key>]`
- 如：`{1 => nil, nil => 3.0, 4+3i => 5/3, 'def' => :define, [1, "3", nil] => :搞咩}`

!!! note
	如此"包容"的映射我还是第一次见

# 5.数组/阵列 #

- 数组元素以`,`分隔，所有数组元素被`[]`包围
- 如：`[1, 2, 3]`
- 引用方式：`<变量>[<索引>]` (索引从0开始)
