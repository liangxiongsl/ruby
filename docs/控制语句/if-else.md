- if
- if-else
- if-else-if
- ternay



###  ###

```rb
/* if */
if(<条件>)
	// ...
end

/* if-else */
if(<条件>)
	// ...
else
	// ...
end

/* if-elsif-else */
if(<条件>)
	// ...
elsif(<条件2>)
	// ...
else(<条件3>)
	// ...
end

/* ternay */
<条件> ? <语句true> : <语句false>
```

!!! tip
	最后一个else可以加入条件
