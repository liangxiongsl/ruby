- 模块是`常量`和`方法`的集合
- 实例方法：包含模块时类中的方法
- 可以在不创建封装对象的情况下调用模块方法，而实例方法则不能
- 模块类似于`类`，但不可实例化，不能创建子类

!!! tip "模块的目的："
	
	- 充当`命名空间`
	- 运行`mixin`工具在类之间共享功能

###  ###

# 1.模块命名空间 #

定义：

```rb
module <模块名>
	<[模块名].方法>
	<常量>
end
```
引用：

```rb
require <模块名>

<模块名>.<方法名>[(<参数列表>)]
<模块名>::<常量>
```

!!! tip
	Ruby常量以大写字母开头

## 2.模块mixins ##

在类定义中使用`include`将模块导入类中


```rb
module A
	def prt
		puts :hello
	end
end
class B
	include A
end

d = B.new
d.prt
```
